!!! note "規則"
    * 分数は(分子) ban (分母)で表す。
    * つまり、(分子) ÷ (分母)。

!!! example "例"
    |Ċatalaŋ|意味|
    |:---|:---|
    |{{r("tan","1")}} {{r("ban","÷")}} {{r("tin","2")}}|$\frac{1}{2}$|
    |{{r("tun","3")}} {{r("ban","÷")}} {{r("ten","4")}}|$\frac{3}{4}$|
    |{{r("ton","5")}} {{r("ban","÷")}} {{r("dan","6")}}|$\frac{5}{6}$|
    |{{r("tin-tin","22")}} {{r("ban","÷")}} {{r("din","7")}}|$\frac{22}{7}$|
    |{{r("tun-ton-ton","355")}} {{r("ban","÷")}} {{r("tan-tan-tun","113")}}|$\frac{355}{113}$|
    |{{r("tan","1")}} {{r("ban","÷")}} {{r("tan-tun-din","137")}}|$\frac{1}{137}$|
