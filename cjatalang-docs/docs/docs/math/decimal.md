!!! note "規則"
    * 小数点はpanで表す。
    * 小数点以下はnon～denの数詞を並べる。

!!! example "例"
    |Ċatalaŋ|意味|
    |:---|:---|
    |tan-pan-tin-tun|1.23|
    |ten-pan-ton-dan|4.56|
    |din-pan-dun-den|7.89|
    |tun-pan-tan-ten|3.14|
    |din-pan-tin-dun|6.28|
    |tan-pan-ten-tan-ten|1.414|
    |tan-pan-dan-tan-dun|1.618|
    |tin-pan-din-tan-dun|2.718|
