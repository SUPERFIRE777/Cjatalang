# 指定詞
この言語には指定詞と言う品詞がある。

これを使うことで、単語に「過去」や「可能」などの意味を付与できる。

!!! abstract "指定詞一覧"
    | カテゴリ | 指定詞 | 意味・用法 |
    | :--- | :--- | :--- |
    | 時間 | pa | 過去 |
    | 時間 | ca | 現在 (通常は省略される) |
    | 時間 | fa | 未来 |
    | その他 | cāca | 推奨 |
    | その他 | ce | 可能 |
    | その他 | ne | 否定 |
    | その他 | nēca | 禁止 |
    | その他 | sa | 強調 |
    | その他 | t̰e | 変化 (直後に te を伴い、t̰e te の形で用いられることが多い) |
    | その他 | wa | 願望 |

!!! note "ポイント"
    * 後ろに意味を追加する対象となる語句を取る。
    * この対象となる語句を「被指定語句」と呼ぶ。
    * te句を取って、{{r("t̰e","変化")}}の変化先や、{{r("wa","願望")}}の対象などを指定することもできる。

!!! example "例文"
    * {{r("Mā","一人称")}} {{r("pa","過去")}} {{r("t̰e","変化")}} {{r("te","→")}} {{r("lica","好む")}} {{r("ċata","ネコ")}}.
    * {{r("Mā","一人称")}} {{r("sa","強調")}} {{r("wa","願望")}} {{r("te","→")}} {{r("jā","二人称")}} {{r("lica","好む")}} {{r("ċata","ネコ")}}.

!!! warning "指定詞の順序に注意"
    指定詞の順序で意味が変わる例がある。
    !!! example "例文"
        1. {{r("Mā","一人称")}} {{r("pa","過去")}} {{r("t̰e","変化")}} {{r("te","→")}} {{r("lica","好む")}} {{r("jā","二人称")}}. 
        2. {{r("Mā","一人称")}} {{r("t̰e","変化")}} {{r("te","→")}} {{r("pa","過去")}} {{r("lica","好む")}} {{r("jā","二人称")}}.
    
    1.は{{r("pa","過去")}} {{r("t̰e","変化")}} {{r("te","→")}}、2.は{{r("t̰e","変化")}} {{r("te","→")}} {{r("pa","過去")}} の順になっている。

    この順序が異なると、{{r("t̰e","変化")}}は異なる変化先を示すことになる。

    1.での変化先は、{{r("lica","好む")}} {{r("jā","二人称")}}だが、2.では{{r("pa","過去")}} {{r("lica","好む")}} {{r("jā","二人称")}}となる。

    これを踏まえて、例の2文を直訳するとこうなる。

    !!! example "直訳"
        1. 私はあなたを好むという状態に変化した。
        2. 私はあなたを好んだという状態に変化する。

    つまり、

    !!! example "意訳"
        1. 私はあなたを好きになった。
        2. 私があなたを好むというのは過去の話になる。

    ということである。
