# 概要
この言語の基本語順はSVOである。

!!! info "特徴"
    * 指定詞による表現
    * 神格化の接尾辞(`-ċat`)がある
    * ネコ専用の代名詞(`ṅā`)がある
    * 演算詞がある

!!! example "例文"
    * {{ r('Mā', '一人称') }} {{ r('lica', '好む') }} {{ r('ċata', 'ネコ') }}.
    * {{ r('Jā', '二人称') }} {{ r('lica', '好む') }} {{ r('ċataċat', 'ネコ(神格化)') }}.
    * {{ r('Hā', '三人称') }} {{ r('lica', '好む') }} {{ r('ṅā', 'ネコ(代名詞)') }}.
    * {{ r('Tan', '1') }} {{ r('lata', '年') }} {{ r('adan', '+(演算詞)') }} {{ r('dan', '6') }} {{ r('mana', '月') }} {{ r('pa', '過去(指定詞)') }} {{ r('suwa', '経過する') }}.
