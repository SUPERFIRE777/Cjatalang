基本はこの数詞たち。

!!! abstract "数詞一覧"
    |数詞|意味|
    |:---|---:|
    |non|0|
    |tan|1|
    |tin|2|
    |tun|3|
    |ten|4|
    |ton|5|
    |dan|6|
    |din|7|
    |dun|8|
    |den|9|
    |don|10|

11以上は、数詞を組み合わせて表現する。

!!! note "規則"
    * 11以上19以下はdon-(1の位)
    * 20以上はnon～denの数詞を並べる。
    * $m \times 10^n$ ($m, n$は整数)は$m$-don-…-don (donが$n$個)も可能
