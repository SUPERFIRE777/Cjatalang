!!! note "規則"
    * 複素数は(実部) adan (虚部) aiで表す。
    * つまり、(実部) + (虚部) i。
    * 実部や虚部が0の場合はその部分は省略可能。
    * 虚部が1のときは虚部は省略可能。

!!! example "例"
    |Ċatalaŋ|意味|
    |:---|:---|
    |{{r("tan","1")}} {{r("adan","+")}} {{r("tin","2")}} {{r("ai","i")}} |$1+2i$|
    |{{r("tun","3")}} {{r("adan","+")}} {{r("ten","4")}} {{r("ai","i")}}|$3+4i$|
    |{{r("ton","5")}} {{r("adan","+")}} {{r("dan","6")}} {{r("ai","i")}}|$5+6i$|
    |{{r("din","7")}} {{r("adan","+")}} {{r("dun","8")}} {{r("ai","i")}}|$7+8i$|
    |{{r("ten","9")}} {{r("ai","i")}}|$9i$|
    |{{r("tun","3")}} {{r("adan","+")}} {{r("ai","i")}}|$3+i$|
    |{{r("ai","i")}}|$i$|
