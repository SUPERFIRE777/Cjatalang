# 文字と発音
この言語の表記には、ラテンアルファベットに加え、いくつかの拡張文字が用いられる。

!!! abstract "文字一覧"
    | 分類 | 文字 | 音素 (IPA) | 備考 |
    | :--- | :--- | :--- | :--- |
    | **ハイフン** | - | /-/ | 単語の連結や特定の形態素境界を示す |
    | **ドット付加文字**<br>(口蓋化) | ċ | /kj/ | |
    | | ṅ | /nj/ | |
    | | ṁ | /mj/ | |
    | **下部修飾文字**<br>(破擦音) | t̰ | /tʃ/ | |
    | **語源的綴り** | ŋ | /n/ | 発音上は n や ng と区別されず、/n/ に最適化される |
    | **長母音** | ā | /aː/ | |
    | | ī | /iː/ | |
    | | ū | /uː/ | |
    | | ē | /eː/ | |
    | | ō | /oː/ | |
    | **通常の子音** | b, d, f, g,<br>h, j, l, m,<br> n, p, s, t,<br> w | 各文字と<br>同等 | 綴り通りに発音される |
    | | c | /k/ | |
    | **通常の母音** | a, e, i, o, u | 各文字と<br>同等 | 綴り通りに発音される |
